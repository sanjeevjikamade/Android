Login with Amazon SDK for Android Readme

The Login with Amazon SDK for Android provides a library and documentation for developers to build connected mobile applications using Login with Amazon. The Login with Amazon SDK for Android supports apps using Android 2.2 or higher (API Version 8).

For more information about using the SDK, please see Getting Started with Login with Amazon for Android Apps
